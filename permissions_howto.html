Create a podcast from an amazon s3 bucket

How to set list permissions so Bucketcast can read your bucket

First, log into the Amazon AWS S3 Console

Then select your bucket, select Properties, and Add Permission in the pane that appears:

Then, select "Everyone" from the dropdown, make sure "List" is checked, and click Save
