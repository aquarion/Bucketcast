Bucketcast

Create a podcast from an amazon s3 bucket

http://altru.istic.net/bucketcast/

Limitations:

You will need to grant "Everyone" the permission "List" in the Amazon AWS S3 Console before this will work, [How do I do that?]
Bucket items have to be publically accessible (obviously)
Trusts file extensions. (Supports mp3/ogg/mp4/aac)
